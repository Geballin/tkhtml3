Hv3 Usage Hints

  • Middle-clicking on the html window changes the status bar from displaying download progress to displaying details of the document node under the cursor. A second click changes things back.
  • Right-clicking on the html window launches the layout debugging tool (a new window).
  • No, there is no "back" button. There's a command entry in the "File" menu instead.
  • To access network files via http, there must be an http proxy running on the localhost accepting connections on tcp port 3128.
